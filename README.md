# Matlab Online Workspace API

### This API exposes MATLAB command line interpreter and language as a backend service

#### You will need:
1. Windows 10
2. MATLAB installation
3. Pyhton3.8+
4. FastAPI + Uvicorn

Download the zip and run `uvicorn main:app` in the root directory.
